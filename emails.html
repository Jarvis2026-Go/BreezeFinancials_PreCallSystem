Setup Guide — How to Automate This

How to Wire This Up (Cheapest Options)

Option A: Calendly + Zapier (Free tier works)

  • Trigger: Calendly "Invitee Created" event
  • Action 1: Send Email 1 immediately via Gmail/Mailgun
  • Action 2: Create a Google Sheet row with booking data
  • For Emails 2 & 3: Use Zapier's "Delay" step (24hr before, 2hr before) → Send email
  • Cost: Free for up to 100 tasks/month on Zapier

Option B: Calendly + Make.com (Free tier, more powerful)

  • Same trigger/action pattern as Zapier
  • Make.com gives you more free operations (1,000/month vs 100)
  • Can also send SMS via Twilio integration for the day-of reminder

Option C: Calendly Workflows (Paid, simplest)

  • Calendly Pro ($12/mo) includes custom email workflows
  • Set up reminder emails directly in Calendly — no code needed
  • Can't fully customize the design, but handles timing automatically

Option D: ConvertKit / Mailchimp (Free tier, most control)

  • Webhook from Calendly → adds subscriber to email automation sequence
  • Full HTML email design control
  • Can tag by industry, revenue range, pain points from intake form
  • Build nurture sequences for no-shows and post-call follow-up too

For SMS reminders (day-of):

  • Twilio: ~$0.01/text, integrate via Zapier/Make.com
  • Calendly Pro: Has built-in SMS reminders
  • Google Voice: Free, but manual (not automated)
